# SCE-Material-2n-Parcial

Aquest repositori conté tots els models d'exemple del 2n parcial de l'assignatura de **Sistemes de Comerç Electrònic** de la Universitat Rovira i Virgili (URV).

## Contingut

- Models d'exàmens amb exercicis pràctics.
- Codis d'exemple en PHP i altres llenguatges.
- Solucions i explicacions detallades.

## Com contribuir

Si trobeu algun error, millora o suggeriment, sou lliures de crear un **issue** al repositori per reportar-ho.

També podeu crear **merge requests** (MR) per proposar canvis directament.
