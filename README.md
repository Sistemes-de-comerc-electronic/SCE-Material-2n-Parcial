# SCE-Material-2n-Parcial

[![License](https://img.shields.io/badge/License-URV-blue.svg)](LICENSE)

Aquest repositori conté tots els models d'exemple del 2n parcial de l'assignatura de **Sistemes de Comerç Electrònic** de la Universitat Rovira i Virgili (URV). Està dissenyat per ajudar els estudiants a preparar-se per a l'examen amb exercicis pràctics, codi d'exemple i solucions detallades.

## 📋 Contingut

- **Models d'exàmens**: Exercicis complets amb enunciats i preguntes típiques del parcial.
- **Codis d'exemple**: Fragments de codi en PHP, Symfony i altres tecnologies relacionades amb el comerç electrònic.
- **Solucions i explicacions**: Respostes detallades amb justificacions basades en principis de DDD, SOLID, seguretat, etc.
- **Fitxers en format .md**: Tots els documents estan en Markdown per facilitar la lectura i processament per part d'intel·ligències artificials (IA), com ara GitHub Copilot o altres eines d'IA, que poden analitzar i generar contingut basat en aquests fitxers de manera eficient.

## 🚀 Com utilitzar

1. **Clona el repositori**:
   ```bash
   git clone https://github.com/Sistemes-de-comerc-electronic/SCE-Material-2n-Parcial.git
   cd SCE-Material-2n-Parcial
   ```

2. **Navega pels fitxers**: Obre els fitxers `.md` amb qualsevol editor de text o visualitzador de Markdown (com VS Code, GitHub, etc.).

3. **Estudia els exemples**: Cada fitxer conté exercicis, codi i solucions. Utilitza les IA per generar resums o preguntes addicionals basades en el contingut.
